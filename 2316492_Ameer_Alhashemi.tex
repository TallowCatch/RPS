\begin{abstract}
This report studies finite-population non-zero-sum RPS dynamics through three connected simulation models. The first model is a well-mixed fixed-size population with Local Update pairwise comparison and explicit parameter sweeps. The second model keeps the same strategic interaction but replaces strategy-switching events with separate birth-only and death-only events, then drives the birth probability with a slow seasonal signal so population size fluctuates strongly over time. The third model moves the interaction process to spatial and network structures and compares periodic lattice behavior with a rewired small-world graph. The analysis is designed to be statistical rather than anecdotal. Every main claim is tied to repeated-seed metrics, distributional summaries, and direct comparisons across models. The most important conclusion is that stability in finite populations is not a single property. It changes with update mechanism, interaction topology, and demographic coupling.
\end{abstract}

\section{Introduction}
Finite-population evolutionary dynamics often diverges from deterministic intuition. In deterministic replicator settings the mixed state can be read directly from phase geometry. In finite populations the same state is influenced by random sampling effects, and those effects do not average out quickly when trajectories move near boundaries. The practical consequence is simple. A single trajectory can look convincing while still being non-representative.

The model class used here follows pairwise comparison Local Update dynamics from the finite-population literature \cite{claussen2016,traulsen2006,nowak2006,claussen2007}. I use the same core game and update philosophy in all three parts so each extension isolates one specific modeling change. This makes interpretation cleaner because observed differences can be linked to demography or structure rather than to a complete change of strategic mechanism.

\section{Model Formulation}
The payoff matrix is
\begin{equation}
A=\begin{pmatrix}
0 & -s & 1 \\
1 & 0 & -s \\
-s & 1 & 0
\end{pmatrix},
\end{equation}
where $s$ controls loss severity relative to gains. Local Update pairwise comparison is implemented through
\begin{equation}
p_{B\to A}=\frac{1}{2}+\frac{w(\pi_A-\pi_B)}{8\pi_{\max}}.
\end{equation}
In the well-mixed finite population I use
\begin{equation}
\pi_{\max}=(N-1)(1+s),
\end{equation}
which is a conservative bound on payoff differences for this game.

This structure keeps the update probability in a valid range once clipped numerically to $[0,1]$. The weak-selection interpretation is preserved by scaling differences through $w$ and by normalizing with a finite payoff bound. The point of this scaling is not merely numerical safety. It also keeps parameter interpretation consistent across sweeps.

\begin{figure}[!htbp]
\centering
\caption{Algorithm 1: Pairwise Comparison Local Update}
\begin{minipage}{0.92\textwidth}
\small
\textbf{Input:} current population state, payoff parameter $s$, selection intensity $w$.\\
1. Sample two distinct agents $A$ and $B$ uniformly at random.\\
2. Compute payoffs $\pi_A$ and $\pi_B$ against all current opponents.\\
3. Compute $p_{B\to A}=\frac{1}{2}+\frac{w(\pi_A-\pi_B)}{8\pi_{\max}}$.\\
4. Clip $p_{B\to A}$ to $[0,1]$.\\
5. With probability $p_{B\to A}$, replace $B$'s strategy with $A$'s strategy.\\
6. Update strategy counts and advance one simulation step.
\end{minipage}
\end{figure}

\section{Experimental Setup and Method}
The three parts are designed as one controlled sequence. I keep the same game and update logic, then introduce one modeling change at a time. This is important for interpretation because it avoids mixing several causes in the same comparison. Every run starts near the mixed state so early transients are comparable across parts.

For Question 1, I use a fixed-size well-mixed population with $N=150$ and horizon $T=30000$. I sweep
\begin{equation*}
s\in\{0.8,1.0,1.2\},\quad w\in\{0.1,0.5,1.0\},
\end{equation*}
with 20 seeds for each pair. This gives nine parameter settings and 180 total runs. The design goal is to see both how payoff asymmetry changes drift pressure and how selection intensity changes adaptation speed.

For Question 2, the pairwise payoff comparison is kept, but the resulting event is no longer direct strategy replacement. Each step still samples two agents and compares their payoffs; the higher-payoff agent is more likely to supply the offspring in a birth event or to survive in a death event. Birth probability is
\begin{equation}
b(t)=b_0+a\sin\left(\frac{2\pi t}{T_{\text{season}}}\right),
\end{equation}
with $b_0=0.50$, $a=0.30$, and $T_{\text{season}}=8000$. I use $T=40000$ and 10 seeds. This forcing creates repeated low-$N$ and high-$N$ windows by construction, so the main question becomes whether composition remains near the mixed state during strong demographic expansion and contraction.

For Question 3, I keep Local Update and change interaction structure. The first extension is a periodic lattice with von Neumann neighborhood. The second extension is a Watts--Strogatz graph with degree eight and rewiring levels $\beta\in\{0,0.1,0.5\}$ \cite{watts1998,szabo2007,reichenbach2007,varga2014}. In the small-world model payoff is normalized by local degree so rewiring comparisons reflect topology and not payoff scale artifacts.

\section{Evaluation Strategy}
The main composition diagnostic is distance from the mixed point:
\begin{equation}
d(t)=\sqrt{\left(\frac{R(t)}{N(t)}-\frac{1}{3}\right)^2+\left(\frac{P(t)}{N(t)}-\frac{1}{3}\right)^2}.
\label{eq:distance}
\end{equation}
This scalar summary lets me compare sustained displacement across parts even when trajectories have different visual shapes. For each run I use late-time means of $d(t)$ to focus on long-horizon behavior.

\subsection*{Theoretical benchmark}
As an infinite-population baseline, the corresponding replicator system is
\begin{equation}
\dot x_i = x_i\big((Ax)_i - x^\top A x\big),\qquad i\in\{R,P,S\},
\end{equation}
with $x_R+x_P+x_S=1$ \cite{nowak2006,hofbauer1998}. The interior equilibrium is
$x^*=(1/3,1/3,1/3)$. For this payoff family, the expected local behavior is inward
flow for $s<1$, neutral cycling around $x^*$ for $s=1$, and outward flow for
$s>1$. Figure~\ref{fig:theory-replicator} shows this deterministic phase-portrait
benchmark and provides context for interpreting finite-population stochastic outcomes.

\begin{figure}[!htbp]
\centering
\includegraphics[width=0.92\textwidth]{results/theory_replicator_phase.png}
\caption{Replicator benchmark in the $(R,P)$ projection for $s=0.8,1.0,1.2$.}
\label{fig:theory-replicator}
\end{figure}

Question 1 uses first-extinction time and edge-contact frequency. First-extinction time captures how long all three strategies coexist before one share hits zero. Edge-contact frequency reports how often runs hit an absorbing boundary before the horizon. This gives a clearer picture than reporting only one seed trajectory.

Question 2 uses demographic spread and composition risk metrics. I report $N_{\min}$, $N_{\max}$, $\bar N$, and $N_{\text{last}}$, together with near-edge exposure and distance statistics. I also compute the seed-level correlation between $N(t)$ and $d(t)$ to check whether demographic contraction is systematically associated with displacement.

Question 3 uses late-time distance, final composition, and clustering. On the lattice, the objective is to characterize finite-size variability under strictly local interactions. On small-world graphs, the objective is to quantify how rewiring and clustering alter displacement from the mixed state.

To strengthen interpretation without changing the scope of the project, I also report selected nonparametric bootstrap percentile intervals from seed-level resampling. These use 20{,}000 resamples with a fixed random seed and are only applied to a small set of summary quantities that drive the main conclusions. The purpose is descriptive uncertainty checking rather than formal hypothesis testing.

\subsubsection*{Observations for $s=0.8$}
The strongest pattern is monotonic growth in median extinction time as $w$ rises, from 9069 at $w=0.1$ to 16848 at $w=1.0$. However, edge-contact frequency is non-monotonic. It is 0.95 at $w=0.1$, reaches 1.00 at $w=0.5$, then drops to 0.80 at $w=1.0$. This indicates that stronger selection can increase persistence time in many runs while still producing a subset of runs that do not hit the boundary by the fixed horizon.

\subsubsection*{Observations for $s=1.0$}
The median extinction time is lowest at $w=0.5$, with value 8426. This is the fastest boundary approach in the full grid. At the same time, mean late distance grows with $w$ and reaches 0.6529 at $w=1.0$, which is the largest late-distance mean in Question 1. The combination suggests that for this payoff balance, stronger selection amplifies displacement even when extinction timing is not monotonic.

\subsubsection*{Observations for $s=1.2$}
For the higher loss severity setting, the median extinction times remain in a narrower band from 10206.5 to 11383.5. Edge-contact is nearly always present, with values 1.00, 1.00, and 0.95 across $w$. The main shift here is not in extinction timing but in sustained displacement, where mean late distance rises from 0.5761 to 0.6208 as $w$ increases.

\subsubsection*{Cross-setting interpretation}
The highest median extinction time appears at $(s,w)=(0.8,1.0)$, while the lowest appears at $(s,w)=(1.0,0.5)$. This gap is large enough to show that no single scalar setting of $s$ or $w$ can be called globally more stable. The combined figure is drawn against $s$ because $s$ is the cleaner organizing axis in this grid, while the effect of $w$ is real but changes across parameter regimes rather than acting as one monotone control.
The bootstrap uncertainty checks in Table~\ref{tab:uncertainty} keep the high-persistence setting centered clearly above the lowest-persistence setting, which supports this ranking while still showing substantial finite-population spread.

\subsection{Question 2: Seasonal demography with separate birth and death}
Question 2 keeps the same game but changes event structure and allows variable population size. The strategic comparison mechanism is unchanged, but its outcome is expressed through payoff-biased birth and death events instead of direct imitation. Table~\ref{tab:q2} shows that this model meets the intended forcing regime. Mean $N_{\min}=50.1$ and mean $N_{\max}=1792.5$ confirm a strong demographic cycle. Mean near-edge exposure is 0.1163, and the mean minimum strategy share is 0.0529.

The seed-level spread is important in this part. Near-edge fraction ranges from 0 to 0.8006 across seeds, so exposure is highly heterogeneous even under the same parameters. Several runs reach $N_{\min}=1$ and end with $N_{\text{last}}=1$, while other runs remain far from collapse. Compared with Question 1, the interpretation changes. In fixed $N$ runs, first-extinction time is a direct stability proxy. In variable $N$ runs, temporary absence and recovery can happen repeatedly, so persistence must be judged through exposure over time and not only through one event timestamp. The average correlation Corr$(N,d)=0.0608$ is small, which means displacement cannot be read from population size alone.
The interval checks in Table~\ref{tab:uncertainty} are consistent with that interpretation. Mean late distance stays well above zero, but the near-edge interval is wide, which matches the seed-level evidence that demographic risk is real and highly heterogeneous rather than uniform across runs.

\subsection{Question 3: Lattice extension and network comparison}
The creative extension keeps Local Update and changes only interaction structure. The lattice model is kept as the main extension, and the small-world analysis is used as a structural comparison under the same update logic. This choice is grounded in the spatial evolutionary-game literature. Reviews of games on graphs emphasize that topology can alter fixation, clustering, and coexistence behavior even when the payoff matrix is unchanged \cite{szabo2007}. For cyclic-dominance systems more specifically, local spatial interactions are known to support persistent pattern formation and biodiversity, while stronger effective mixing can weaken that stabilizing role \cite{reichenbach2007,varga2014}. The small-world comparison therefore has a clear literature motivation rather than being only a technical add-on.

\section{Integrated Interpretation}
Taken together, the three questions show that coexistence in finite populations is conditional and mechanism-dependent. The fixed-size sweep demonstrates that parameter effects are not globally monotonic, so interpretation must be based on distributions rather than single trajectories. The seasonal model demonstrates that demographic forcing changes what stability evidence is meaningful, because temporary strategy loss can appear without immediate terminal collapse. The structural extension demonstrates that network organization can shift both average displacement and variability even when the update rule is unchanged.

The most consistent pattern across all parts is that no single metric is sufficient. Extinction timing, edge-contact exposure, late-time displacement, and structural summaries each reveal a different aspect of finite-population behavior. The report therefore treats claims as multi-metric and comparative, which is closer to how these stochastic systems should be evaluated in practice.

\section{Critical Reflection}
The analysis is stronger than a trajectory-only writeup, but several limits remain and affect interpretation. The first limit is finite horizon dependence because metrics such as late-time means and first-extinction times can shift when the observation window changes. The second limit is parameter scope because the chosen values for $s$, $w$, and seasonal forcing are informative but still represent only one region of the model space. The third limit is inferential scope because, although selected bootstrap confidence intervals are now reported for key summaries, the study still does not perform formal hypothesis tests or exhaustive uncertainty analysis across the full parameter space. The fourth limit is model scope because Local Update is only one evolutionary rule, so broader conclusions about cyclic dominance would need direct replication under alternative rules. Even with these limits, the central comparisons are internally consistent, reproducible from the scripts, and supported by repeated-seed evidence rather than isolated runs.

\section{Conclusion}
This report provides a complete finite-population RPS analysis across three linked models with one consistent strategic core. Question 1 shows that stability claims depend on which statistic is used and cannot be reduced to one trajectory. Question 2 shows that seasonal demography creates strong risk heterogeneity and changes how persistence should be measured. Question 3 shows that interaction structure alone can shift both displacement and variability while preserving the same game and update process. The overall takeaway is that credible conclusions in finite-population evolutionary dynamics require explicit comparisons, repeated seeds, and metric choices that match the mechanism under study.
